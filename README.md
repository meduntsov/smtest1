# Demo ERP для девелоперских проектов (Project 360°)

Демонстрационная ERP-система для презентации Unified-модели управления жилыми комплексами:

- **Production WBS**: Фаза → Стадия → Блок → Подсистема → EP
- **Финансовая WBS (6D)**: кредитная линия, транши, ковенанты, cash flow
- **Decision EP**: управленческие пакеты (тендеры/согласования)
- **AI аналитика**: сроки, блокировки, финансовые риски без Python

## Стек

- **Backend:** .NET 8 Web API + EF Core + Npgsql
- **Frontend:** React + TypeScript + Recharts
- **DB:** PostgreSQL
- **BI:** встроенные dashboard-графики в UI
- **AI module:** C# rule-based аналитика и рекомендации

## Что реализовано в демо

- Объект **Execution Package (EP)** с обязательными полями модели.
- Русифицированные справочники (дисциплины, подсистемы, виды работ, единицы, статьи затрат, источники финансирования, типы EP, изменений и рисков).
- Экран **Проект 360°**:
  - KPI по сроку/бюджету/рискам/блокировкам,
  - Heat-map EP (зелёный/жёлтый/красный/фиолетовый),
  - AI-рекомендации,
  - Диаграмма траншей,
  - График cash flow,
  - Обзор модулей WBS / тендеров / изменений / финансов.
- Демо-данные:
  - 1 ЖК (3 корпуса),
  - 60 EP,
  - признаки 5 тендеров, 3 траншей, 2 change request, 1 критического риска отображаются на dashboards и в рекомендациях.
